C++ Challenge: OOP

Challenge Problem: Bank Account Management System

Design and implement a simple bank account management system using Object-Oriented Programming (OOP) principles in C++. The system should allow users to create bank accounts, deposit money, withdraw money, and check their account balance.

Here's a basic outline of the classes you might need:

1. Account

Represents a bank account. It should have properties like account number, account holder name, account type, and balance. Include methods to deposit, withdraw, and check balance.

2. SavingsAccount

Inherits from the Account class. Represents a savings account that earns interest over time. You can add additional properties and methods specific to savings accounts, such as interest rate.

3. CheckingAccount

Inherits from the Account class. Represents a checking account. You can add additional properties and methods specific to checking accounts, such as overdraft limit.

4. Bank

Represents a bank that manages multiple accounts. Include methods to create new accounts, close accounts, and search for accounts by account number or account holder name.

You can also include error handling to handle cases like insufficient balance for withdrawals, invalid account numbers, etc.

Challenge yourself by implementing additional features like transaction history, account locking, or account types.
