C++ Challenge: Intermediate Programming

Take on the challenge to demonstrate your understanding of programming in C++.

Challenge Problem: Bank Account Management System

Design a simple banking system in C++ that allows users to create bank accounts, deposit money, withdraw money, and check their balance.

Requirements:

1. Implement a class named 'BankAccount' with the following functionalities:

Constructor to initialize the account holder's name, account number, and balance.

Functions to deposit money, withdraw money, and check balance.

Ensure that the balance cannot be negative.

Implement appropriate error handling for invalid transactions (e.g., insufficient balance).

2. Implement a class named 'Bank' to manage multiple bank accounts:

Maintain a collection of 'BankAccount' objects.

Allow users to create new bank accounts.

Allow users to deposit money, withdraw money, and check balance for a specific account.

Ensure that account numbers are unique across the bank.

3. Implement a simple user interface in the main function:

Allow users to interact with the banking system by choosing options from a menu.

4. Ensure that the program is well-documented with comments explaining the purpose and functionality of each class and function.

5. Test the program thoroughly to ensure correctness and handle edge cases gracefully.
